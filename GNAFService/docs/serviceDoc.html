Documentation

Brief description

This SOAP service provides 3 different search functions, which search the Australian Government released PSMA Geocoded National Address File (G-NAF) Database.

Link to the WSDL document of the service

http://localhost:8080/GNAFService/GNAFAddressingService?wsdl

addressExistsInFile( )

The input for this is comma separated and follows the format street number,street name,suburb name, state,postcode.

Input 1

Field Name Type Value
StreetNumber String 17,billabong street,lawson,act,2617

Output 1

Found Address: 17,billabong street,lawson,act,2617

Input 2

Field Name Type Value
StreetNumber String 4570,BLACK STUMP WAY,COOLAH,NSW,2843

Output 2

Found Address: 4570,BLACK STUMP WAY,COOLAH,NSW,2843


similarAddressesInFile( )

Street Number and Street Name are REQUIRED. SOAP Faults are presented if they are not present.

SuburbName String Mascot

Errors

Error Code Description
ERROR 0 There are no entries are found in the Database corresponding to the input.
ERROR 1 Mandatory input fields are empty
ERROR 2 Input values are not the right format. (e.g., postcodes are numbers, state names are characters, etc.)
